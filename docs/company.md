# Sthololwazi Projects — Company Context

Sthololwazi Projects is a turnkey construction and infrastructure contractor delivering projects from concept to key handover.

Primary services (summary):
- Planning & Pre-Construction (feasibility, BOQs, scheduling)
- Architectural Services (plans, elevations, as-built)
- Engineering Services (structural, foundations, drainage)
- Construction disciplines (earthworks, concrete, brickwork, roofing, finishes)
- MEP (plumbing, electrical)
- Project management, procurement, testing & commissioning, handover

This repository (SPDS) builds software to support Sthololwazi's workflows: project setup, building modelling, documentation generation (drawings, BOQs, specifications), standards compliance, and eventual AI-assisted design and operations.
